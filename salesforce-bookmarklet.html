How to use

  1. 1Drag the tool above to your bookmarks bar
  2. 2Go to the page relevant to the tool you'd like to use
  3. 3Click the bookmark to trigger your automation
  4. 4See the response in a popup window

What it sends

{ "opportunityId": "006S700000FIMsgIAH", "sourceUrl": "https://celonis.lightning.force.com/lightning/r/Opportunity/006S700000FIMsgIAH/view", "timestamp": "2025-07-02T15:30:00.000Z" }

Features

  • Automatically extracts Opportunity IDs from URLs
  • Routes through AWS API Gateway for Salesforce CSP compliance
  • Shows formatted JSON responses in a popup
  • Auto-closes after 30 seconds
  • Works on any Salesforce org

Security Features

  • Uses AWS API Gateway as secure proxy
  • API credentials managed server-side in AWS Lambda
  • Complies with Salesforce Content Security Policy
  • All requests use HTTPS encryption

Architecture

Salesforce → AWS API Gateway → AWS Lambda → Make.com Webhook
